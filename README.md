# RecordMate

RecordMate is a Raspberry Pi powered vinyl companion.

It recognizes vinyl records through audio fingerprinting, displays album artwork and metadata, and can synchronize playback with Spotify.

## Planned features

- Automatic vinyl recognition
- Spotify synchronization
- Album artwork display
- Now Playing screen
- Idle clock and weather screen
- Listening history
- Discogs integration

## Tech stack

- Raspberry Pi OS
- Python
- Flask
- Spotify Web API
